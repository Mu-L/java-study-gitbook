                                                                                          # Redis缓存预热

                                                                                          # 1. 简介

                                                                                          缓存预热就是系统上线后，将相关的缓存数据直接加载到缓存系统

                                                                                          这样可以避免在用户请求的时候，先查询数据库，然后再将数据缓存的问题！用户直接查询事先被预热的缓存数据！

                                                                                          # 2.解决方案

                                                                                          # 2.1 缓存刷新页面

                                                                                          直接写个缓存刷新页面，上线时手工操作

                                                                                          # 2.2 项目启动自动加载

                                                                                          数据量不大，可以在项目启动的时候自动进行加载

                                                                                          目的就是在系统上线前，将数据加载到缓存中。
